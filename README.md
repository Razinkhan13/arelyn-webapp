# ARELYN Web Experience

ARELYN is a modern female fashion brand experience built with React + Vite. The single-page layout highlights the latest Atelier drop, featured capsule pieces, and curated collections. All remote imagery automatically swaps to an inline fallback illustration if loading fails, ensuring the page always renders with contextual visuals.

## Getting started

```bash
npm install
npm run dev   # start local dev server
npm run build # production build
npm run lint  # lint with ESLint
```

The template does not ship with a test runner script by default.

## Notes

- Imagery uses remote, license-friendly photography; each `<img>` has a built-in fallback to an inline SVG to keep the layout intact if something doesn’t load.
- The layout is responsive for mobile and desktop, with brand-focused typography and gradients to match the ARELYN aesthetic.
